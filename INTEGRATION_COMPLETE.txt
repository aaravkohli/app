## How It Works

1. User types prompt in frontend
2. Frontend sends HTTP request to API
3. API calls backend analysis
4. Backend checks for threats (ML + patterns)
5. If safe: Generates AI response
6. If dangerous: Returns block reason + suggestion
7. Frontend displays results with metrics
